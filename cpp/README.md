# clankeRS C++ SDK

Idiomatic C++17 wrappers over the stable C ABI (`clankers-ffi`). Link this
library from robot applications, ROS 2 nodes, or vendor SDKs without writing Rust.

## Build

From the repository root:

```bash
bash scripts/build_cpp_sdk.sh
```

Or manually:

```bash
cargo build -p clankers-ffi --release
cmake -B cpp/build -S cpp
cmake --build cpp/build
```

## Quick example

```cpp
#include <clankers/clankers.hpp>

auto engine = clankers::Engine::builder()
                  .backend("onnxruntime")
                  .model_path("model.onnx")
                  .build();

std::vector<float> input(4, 0.5f);
auto view = clankers::TensorView::from_external(
    {reinterpret_cast<const std::byte*>(input.data()), input.size() * sizeof(float)},
    clankers::DType::F32,
    clankers::Shape({1, 4}));

auto outputs = engine.run({view});
```


## Ownership

| Object | Owner | Free with |
|--------|-------|-----------|
| `clankers::Engine` | C++ RAII | destructor |
| `clankers::Tensor` | C++ RAII | destructor |
| External `TensorView` buffer | **Caller** | n/a |
| `run_into` output buffer | **Caller** | n/a |

See `cpp/include/clankers/clankers.h` for the full C ABI reference (generated by `cbindgen`).
